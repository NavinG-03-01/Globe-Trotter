# Project

## Project info

This is a starter project scaffold. Edit files locally or via your preferred web editor.

## How can I edit this code?

You can edit the project locally using your IDE or an online editor.

Requirements: Node.js and npm (use nvm to install if needed).

Quick start:

```sh
git clone <YOUR_GIT_URL>
cd <YOUR_PROJECT_NAME>
npm install
npm run dev
```

## What technologies are used for this project?

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## How can I deploy this project?

Follow your hosting provider's instructions to build and deploy the Vite app.

## Custom domain

Configure your hosting provider's domain settings to point to the deployed site.
